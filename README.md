# lua-fundamentals
A beginner-friendly repository for learning the core concepts of the Lua programming language. This project covers Lua fundamentals with simple explanations, practical examples, and hands-on practice programs to help you build a strong foundation in scripting and programming.


🚀 Topics Covered
Introduction to Lua
Variables & Data Types
Operators
Conditional Statements
Loops
Functions
Tables
Strings
Arrays
File Handling
Modules
Object-Oriented Basics
Error Handling


💡 Features
Clean and beginner-friendly code
Well-structured examples
Easy-to-understand explanations
Practice programs for each topic
Perfect for students and beginners


🎯 Purpose
This repository is created for anyone who wants to learn Lua from scratch and improve their programming fundamentals through practical coding examples.


🛠️ Technologies Used
Lua Programming Language


📚 Learning Outcome
By completing this repository, you will:

Understand Lua syntax and fundamentals
Write basic to intermediate Lua programs
Learn problem-solving using Lua
Build a strong base for game development and scripting
